# Prasad Pawar Portfolio

Personal portfolio website for Prasad Pawar, focused on software engineering, AI/data science projects, certificates, and contact links.

Live site: [prasadpawarz.github.io](https://prasadpawarz.github.io)

## Overview

This repository contains a static portfolio built with HTML, CSS, and JavaScript. It is designed to present:

- Featured software engineering and AI/data projects.
- A complete project library with category filters.
- Project detail pages with screenshots, galleries, and README-style notes.
- A featured certificate section plus a full certificate library.
- Education, internship experience, skills, and contact information.

## Pages

| Page | Purpose |

## Run Locally

No build step is required.

```bash
git clone https://github.com/PrasadPawarZ/PrasadPawarZ.github.io.git
cd PrasadPawarZ.github.io
```

Then open `index.html` in a browser.

For best local testing, use a simple static server:

```bash
python -m http.server 8000
```

Open:

```text
http://localhost:8000
```

## Deployment

This repository is deployed with GitHub Pages from the `main` branch.

After pushing changes, GitHub Pages may take a minute or two to update.
